NetDigest

Network → tokens

A DevTools panel that turns noisy network traffic into a compact, secret-free TOON digest — small enough to paste into an LLM, complete enough to debug with.

DevTools — NetDigest

This is the real extension, running live on scripted demo traffic — click rows, switch S/M/L, try the API map, copy the TOON.

×12 Dedup & diff

Identical calls merge into one entry; array items after the first only show what changed. The shape once, never twice.

*** Secrets redacted

Passwords, tokens, API keys and JWT-looking values are stripped at capture time. Auth headers never reach memory.

S·M·L Budget control

Three detail levels, a live token counter, and per-endpoint overrides when one response deserves more.

⚑ REC Flow recorder

Clicks, form submits and SPA navigations become timeline steps — the model sees what triggered every request.

{ } API map

One click condenses the traffic into a normalized endpoint contract: methods, statuses, query keys, example shapes.

src/ Initiators

Source maps resolve every call back to the hook that fired it — useArticles.useQuery, not a hashed chunk.
